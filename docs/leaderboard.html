Reproduce these results

All benchmarks are open-source and part of the main repository. Run them yourself to verify the numbers.

# Run the full task-based simulation
uv run python benchmarks/run_benchmark.py

# Run the issues benchmark (requires API keys)
uv run python benchmarks/run_benchmark.py --issues-file benchmarks/issues.json

# Run SWE-Bench Lite (requires Docker + Anthropic key)
uv run python benchmarks/swe_bench/run.py eval
