# 609 — Founder Content Calendar

**Role:** docs
**Priority:** 1 (critical)
**Scope:** small
**Depends on:** none

## Problem

There is no content strategy for building Alex Chernysh's personal brand as a multi-agent orchestration expert. Harrison Chase (LangChain) and Joao Moura (CrewAI) built their frameworks through consistent personal content. The founder IS the brand for developer tools.

## Design

Establish a content calendar: 2x/week Twitter/X posts (technical insights, progress updates, hot takes on agent tooling), 1x/week blog post (deeper technical content, benchmarks, architecture decisions), 1x/month conference talk submission. Create a content bank of 20+ topic ideas spanning: Bernstein development updates, multi-agent orchestration patterns, agent tooling landscape analysis, and production lessons. Define content pillars: deterministic orchestration, cost governance, provider-agnostic agents. Set up a simple tracking system (markdown file) for planned/published content.

## Files to modify

- `docs/content-calendar.md` (new)
- `docs/content-topics.md` (new)

## Completion signal

- Content calendar exists with 3-month schedule
- 20+ topic ideas documented with outlines
- Content pillars defined and documented
