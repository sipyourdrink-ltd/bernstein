# 336b — Token Growth Monitor with Auto-Intervention
**Role:** backend  **Priority:** 1 (critical)  **Scope:** small

## Problem
"Quadratic token growth is the most dangerous economic trap." 90% cost reduction possible with context compression.

## Design
Monitor tokens per agent over time. Alert on quadratic growth. Auto-kill agents using >50K tokens with no file changes. Context compression: strip unused tool definitions, summarize conversation history.
