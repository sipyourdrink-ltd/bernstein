# 603 — Coordinated Launch Plan

**Role:** docs
**Priority:** 1 (critical)
**Scope:** medium
**Depends on:** #608, #610

## Problem

There is no launch plan for Bernstein's public debut. Uncoordinated launches waste momentum. A coordinated multi-platform launch within a 48-hour window maximizes initial visibility and creates the appearance of organic buzz.

## Design

Create a coordinated launch plan covering HN Show, Reddit (r/LocalLLaMA, r/programming, r/MachineLearning), Twitter/X, and Product Hunt within a 48-hour window. Build to 50-100 GitHub stars from personal network before going public. Prepare a demo video (3 min max) showing the "wow" moment of multi-agent orchestration. Write platform-specific copy for each channel (HN is technical/understated, Reddit is community-focused, Twitter is visual/punchy). Plan to respond to every HN comment within the first 6 hours. Schedule posts for optimal timezone coverage. Include a pre-launch checklist: README polished, quickstart works, demo video uploaded, comparison pages live.

## Files to modify

- `.sdd/decisions/launch-plan.md` (new)
- `docs/launch/` (new directory with platform-specific copy)

## Completion signal

- Launch plan document exists with timeline, platform copy, and pre-launch checklist
- Demo video script/storyboard included
- Response strategy documented for each platform
